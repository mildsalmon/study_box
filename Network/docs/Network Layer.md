# 네트워크 계층 (Network Layer) (=우체국) - 3계층

> 직접적으로 통신과 연결되어 있음

- Network Core의 끝단간 전송을 구현하기 위한 경로설정.
	- 양 끝단으로의 전달을 책임짐. 
	- 패킷을 끝에서 끝으로 경로를 설정하여 전달을 책임짐.
- 스위치 (교환기)가 관여함.
