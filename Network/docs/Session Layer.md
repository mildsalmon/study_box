# 1. 세션 계층 (Session Layer) - 5계층

- 접속을 하면, 접속이 끝날때까지 세션을 유지시키는 것.

- 객체 교환
- 대표적인 예
	- FTP, HTTP
		- 역사적으로 FTP가 먼저 나왔지만, 현재 주로 사용하는건 HTTP
