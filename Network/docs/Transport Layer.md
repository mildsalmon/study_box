# 1. 전송 계층 (Transport Layer) (=비서) - 4계층

> 직접적으로 통신과 연결되어 있음

- 양 끝단의 전송 품질보장
	- 네트워크는 잡음과 혼잡으로 인해 패킷이 사라질 수 있다.
	- 이때, 모든 패킷이 전송됨을 보장함.
- 신뢰성있는 네트워크의 품질을 제공
- 밑(물리, 데이터링크, 네트워크 계층)에가 부실해도 재전송을 통해서 신뢰성을 보장할 수 있다.
